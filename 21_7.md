## - 수정사항 및 결과
>> 그래픽

1. 엔딩화면 제작 -> 100%

2. 선택지별 따른 엔딩 화면 러프 및 선화 -> 90% 
(러프 - 완, 선화 - 진행 중)

3. 백설공주 스크립트 -> 0%

4. 퍼즐화면 수정 -> 100%



>> 프로그래밍

1. 전체 오류 수정 -> 90%

오류 텍스트는 전부 수정, 스토리 부분 따로 실행시 유니티 작동이 멈춤

2. 스토리(동화) 스크립트 수정 -> 50% 

3. 엔딩 씬 수정 -> 50% (4개중 2개 완성)


>> 공동

1. 깃허브 수정 -> 80%

2. 엔딩 스토리 수정 -> 50%
